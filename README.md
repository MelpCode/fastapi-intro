# fastapi-intro

## Start the project

### 1. Activate mongodb service

```
  > mongod
```

### 2. Activate the virtual environment
```
  > .\venv\Scripts\activate
```
***Note: deactivate virtual enviroment***
```
  > .\venv\Scripts\deactivate
```

### 2. start fastapi project
```
  (venv)> uvicorn app:app --reload
```
